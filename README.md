## Why

Building a RAG system is easy. Knowing if it's any good is hard. `rag-eval` runs your RAG endpoint against a labeled eval-set and reports three numbers: did it retrieve the right docs, did it stay faithful to them, did it answer correctly.

Extracted from production work on multi-tenant RAG SaaS.
